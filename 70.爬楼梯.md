## 思路（必填）

1. 确定dp数组以及下标的含义。dp[i]： 爬到第i层楼梯，有dp[i]种方法
2. 确定递推公式：dp[i] = dp[i - 1] + dp[i - 2]
3. dp数组如何初始化：不考虑dp[0]如何初始化，只初始化dp[1] = 1，dp[2] = 2，然后从i = 3开始递推，这样才符合dp[i]的定义。
4. 确定遍历顺序：从前向后遍历
5. 举例推导dp数组

## 代码

```java
class Solution {  
    public int climbStairs(int n) {  
        if(n <= 1) return n;  
        int[] dp = new int[n + 1];  
        dp[1] = 1;  
        dp[2] = 2;  
        for (int i = 3; i <= n; i++) {  
            dp[i] = dp[i - 1] + dp[i - 2];  
        }  
        return dp[n];  
    }  
}
```
