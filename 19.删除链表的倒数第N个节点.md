![[19.png]]

## 思路（必填）

1. 双指针的经典应用，如果要删除倒数第n个节点，让fast移动n步，然后让fast和slow同时移动，直到fast指向链表末尾。删掉slow所指向的节点就可以了。
2. 但为了方便，我们让fast首先走n + 1步 ，为什么是n+1呢，因为只有这样同时移动的时候slow才能指向删除节点的上一个节点，方便做删除操作。

## 代码

```java
class Solution {  
    public ListNode removeNthFromEnd(ListNode head, int n) {  
        ListNode dummy = new ListNode();  
        dummy.next = head;  
        ListNode fast = dummy;  
        ListNode slow = dummy;  
        //让fast先走n+1步，方便做删除操作，slow指向要删除节点的上一个节点  
        for (int i = 0; i < n + 1; i++) {  
            fast = fast.next;  
        }  
        while (fast != null) {  
            fast = fast.next;  
            slow = slow.next;  
        }  
        if (slow.next != null) {  
            slow.next = slow.next.next;  
        }  
        return dummy.next;  
    }  
}
```
