## 题目
![[19.png]]

## 思路（必填）
1. 创建一个虚拟头结点 `dummy`，将 `head` 连接到它。
2. 让 `fast` 指针先走 `n+1` 步。  
3. 然后同时移动 `fast` 和 `slow` 指针，直到 `fast` 到达链表末尾。  
4. 此时 `slow` 指针指向的就是要删除节点的前一个节点，进行删除操作。  

## 关键点
>[!Tip] 采用快慢指针法，先让快指针走 `n+1` 步，然后快慢指针一起走，当快指针到达末尾时，慢指针指向的就是要删除节点的前一个节点。
1. 双指针的经典应用，如果要删除倒数第`n`个节点，让`fast`移动`n`步，然后让`fast`和`slow`同时移动，直到`fast`指向链表末尾。删掉`slow`所指向的节点就可以了。
2. 但为了方便，我们让`fast`首先走`n + 1`步 ，为什么是`n + 1`呢，因为只有这样，同时移动的时候`slow`才能指向删除节点的上一个节点，方便做删除操作。

## 代码

```java
class Solution {  
    public ListNode removeNthFromEnd(ListNode head, int n) {  
        ListNode dummy = new ListNode();  
        dummy.next = head;  
        ListNode fast = dummy;  
        ListNode slow = dummy;  
        //让fast先走n+1步，让 fast 和 slow 之间保持一个 n 的距离，slow指向要删除节点的上一个节点  
        for (int i = 0; i < n + 1; i++) {  
            fast = fast.next;  
        }  
        // 同时移动fast和slow指针，直到fast指针到达链表末尾  
        while (fast != null) {  
            fast = fast.next;  
            slow = slow.next;  
        }  
        if (slow.next != null) {  
            slow.next = slow.next.next;  
        }  
        return dummy.next;  
    }  
}
```

## 复杂度

- Time: O(n)
- Space: O(1)
