## 题目

## 思路（必填）
### 问题分析
给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子下雨之后能接多少雨水。
### 算法选择
使用双指针法（最优解）。核心洞察是：
  - 每个位置能接的雨水 = min(左边最高柱子, 右边最高柱子) - 当前柱子高度
  - 不需要预先知道左右所有的最大值，只需知道左右边界的最大值即可
### 实现步骤
  1. 初始化左右指针分别指向数组两端
  2. 维护 preMax 和 sufMax 记录左右两边遇到的最大高度
  3. 每次比较 preMax 和 sufMax：
    - 较小的一侧可以确定该侧指针位置的蓄水量（因为另一侧有更高的柱子挡着）
    - 计算蓄水量并移动指针
  4. 直到左右指针相遇

## 关键点
>[!Tip] 
>- 为什么移动较小边界的指针？
>- 当 preMax < sufMax 时，left 位置的蓄水量只取决于 preMax（右边有更高的保障）
>- 这样可以避免使用额外数组存储每个位置的左右最大值

## 代码

```java
class Solution {  
    public int trap(int[] height) {  
        int ans = 0;  
        int preMax = 0; // 前缀最大值，随着左指针 left 的移动而更新  
        int sufMax = 0; // 后缀最大值，随着右指针 right 的移动而更新  
        int left = 0;  
        int right = height.length - 1;  
  
        while (left < right) {  
            preMax = Math.max(preMax, height[left]);  
            sufMax = Math.max(sufMax, height[right]);  
            if (preMax < sufMax) {  
                ans += preMax - height[left];  
                left++;  
            } else {  
                ans += sufMax - height[right];  
                right--;  
            }  
        }  
  
        return ans;  
    }  
}
```

## 知识点
双指针优化
在暴力解法中，我们可以看到只要记录左边柱子的最高高度 和 右边柱子的最高高度，就可以计算当前位置的雨水面积，这就是通过列来计算。

当前列雨水面积：min(左边柱子的最高高度，记录右边柱子的最高高度) - 当前柱子高度。

为了得到两边的最高高度，使用了双指针来遍历，每到一个柱子都向两边遍历一遍，这其实是有重复计算的。我们把每一个位置的左边最高高度记录在一个数组上（maxLeft），右边最高高度记录在一个数组上（maxRight），这样就避免了重复计算。

当前位置，左边的最高高度是前一个位置的左边最高高度和本高度的最大值。
即从左向右遍历：`preMax = max(preMax, height[left])`；
从右向左遍历：`sufMax = max(sufMax, height[right])`。

- Time: O(n)
- Space: O(1)
