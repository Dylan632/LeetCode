## 思路（必填）
### 问题分析
给定一个不含重复元素的整数数组 `nums`，返回所有可能的子集（幂集）。要求子集不能重复，且顺序不重要。
### 算法选择
使用回溯法（Backtracking），通过递归枚举每个元素的“选”或“不选”决策。
### 实现步骤
1. 初始化结果列表 `ans` 和当前路径 `path`
2. 从索引 0 开始 DFS 递归
3. 不选当前元素：先将当前 `path` 加入结果（空集也是子集）
4. 选当前元素：从当前位置开始遍历，选择 `nums[j]`，递归进入下一层
5. 回溯：递归返回后撤销选择，尝试其他可能性

## 关键点

> [!tip]
> - 通过 `dfs(j + 1, ...)` 确保每个元素只被使用一次，避免重复子集
> - `ans.add(new ArrayList<>(path))` 必须复制 `path`，否则后续修改会影响已添加的结果

## 代码

```java
class Solution {  
    public List<List<Integer>> subsets(int[] nums) {  
        List<List<Integer>> ans = new ArrayList<>();  
        List<Integer> path = new ArrayList<>();  
        dfs(0, nums, path, ans);  
        return ans;  
    }  
  
    private void dfs(int i, int[] nums, List<Integer> path, List<List<Integer>> ans) {  
        // 不选，把当前子集加入答案  
        ans.add(new ArrayList<>(path));  
        // 选，枚举选择的数字  
        for (int j = i; j < nums.length; j++) {  
            path.add(nums[j]);  
            // 选 nums[j] 意味着 i 到 j-1 都跳过不选，下一个数从 j+1 开始选  
            dfs(j + 1, nums, path, ans);  
            path.removeLast();  
        }  
    }  
}
```

## 知识点
```
dfs(0)                                                 path = [], ans = [[]]
├── for j=0, 选 1: path.add(1)                         path = [1]
│   ├── dfs(1)                                         ans = [[], [1]]
│   │   ├── for j=1, 选 2: path.add(2)                 path = [1, 2]
│   │   │   ├── dfs(2)                               ans = [[], [1], [1, 2]]
│   │   │   │   └── for j=2, 不满足 j<2, return        循环不执行，直接返回
│   │   │   └── path.removeLast() (回溯: 弹出 2)       path = [1]
│   │   └── for 循环结束, return
│   └── path.removeLast() (回溯: 弹出 1)               path = []
├── for j=1, 选 2: path.add(2)                         path = [2]
│   ├── dfs(2)                                  ans = [[], [1], [1, 2], [2]]
│   │   └── for j=2, 不满足 j<2, return                循环不执行，直接返回
│   └── path.removeLast() (回溯: 弹出 2)               path = []
└── for 循环结束, return                               结束整个查找过程
```

## 复杂度

- **Time**: $O(n \cdot 2^n)$，其中 $n$ 为 `nums` 的长度。答案的长度为子集的个数，即 $2^n$，同时每次递归都把一个数组放入答案，因此会递归 $2^n$ 次，再算上加入答案时复制 `path` 需要 $O(n)$ 的时间，所以时间复杂度为 $O(n \cdot 2^n)$。

- **Space**: $O(n)$，返回值的空间不计。
