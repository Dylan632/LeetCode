## 问题分析

- 输入数组已按非递减顺序排序（可能包含负数）
- 要求返回每个元素平方后按非递减顺序排列的新数组
- 关键洞察：负数平方后可能变成最大值，所以简单的平方后再排序不是最优解

## 思路

1. 初始化左右指针分别指向数组两端
2. 创建结果数组，从后往前填充（先填大的）
3. 比较两端元素平方值，较大者放入结果数组末尾
4. 移动相应指针，继续比较直到处理完所有元素


## 关键点

> [!Tip]
> - **逆向填充**：从后往前填充结果数组，避免了数组元素移动
> - **利用有序性**：原数组有序，两端必然存在最大值，故用双指针从两端收缩

## 代码

```java
class Solution {  
    public int[] sortedSquares(int[] nums) {  
        int left = 0, right = nums.length - 1;  
        int[] result = new int[nums.length];  
        //从后往前填新的空数组  
        int index = nums.length - 1;  
        while (left <= right) {  
            if (nums[left] * nums[left] > nums[right] * nums[right]) {  
                //index每次用完需要-1，相当于新数组需要填的空位往前移动1位  
                result[index--] = nums[left] * nums[left];  
                left++;  
            } else {  
                result[index--] = nums[right] * nums[right];  
                right--;  
            }  
        }  
        return result;  
    }  
}
```

## 知识点

- **双指针**：从两端向中间收缩，适用于有序数组
- **逆向填充**：当结果顺序与处理顺序相反时，可反向填充避免元素移动

## 复杂度

- Time: O(n)
- Space: O(n)
