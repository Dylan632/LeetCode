## 题目
![[238.png]]

## 思路（必填）
### 算法选择
前缀积 + 后缀积，两次线性扫描。
### 实现步骤
1. 从右到左计算“右侧乘积”放入结果数组。
2. 从左到右维护“左侧乘积”并乘到结果数组。
3. 返回结果数组。

## 关键点
>[!Tip] 用一个滚动变量维护前缀积，避免额外数组。

## 代码

```java
class Solution {  
    public int[] productExceptSelf(int[] nums) {  
        int n = nums.length;  
        int[] ans = new int[n];  
        // 最后一个位置右边没有元素，所以设为 1        ans[n - 1] = 1;  
        // 求后缀积  
        for (int i = n - 2; i >= 0; i--) {  
            ans[i] = ans[i + 1] * nums[i + 1];  
        }  
        // 此时 ans[i] = nums[i+1] * nums[i+2] * ... * nums[n-1]  
        // 再乘上前缀积  
        // prefix 表示当前位置左边所有乘积  
        int prefix = 1;  
        for (int i = 0; i < n; i++) {  
            // 左前缀 * 右后缀  
            ans[i] = prefix * ans[i];  
            // 更新前缀积  
            prefix *= nums[i];  
        }  
        return ans;  
    }  
}
```
