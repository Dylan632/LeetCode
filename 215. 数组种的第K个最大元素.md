## 思路（必填）
### 问题分析
在无序数组中找到第 k 大的元素，要求时间复杂度优于排序的 O(n log n)。

### 算法选择
选择**快速选择算法**，基于快速排序的分区思想：
- 平均时间复杂度 O(n)，每次排除一半元素
- 空间复杂度 O(log n) ~ O(n)，取决于实现方式

### 实现步骤
1. 转换问题：第 k 大 = 第 (n - k + 1) 小（升序排序后）
2. 随机选 pivot：避免有序数组的 O(n²) 最坏情况
3. 三分区：将数组分为大于、等于、小于 pivot 的三部分
4. 递归缩小范围：
   - k 在大区：递归 big
   - k 在小区：递归 small（调整 k 值）
   - k 在等区：直接返回 pivot

## 关键点
> [!Tip]
> - 随机化 pivot 确保平均 O(n) 性能
> - 三分区正确处理重复元素
> - 递归时正确调整 k 的值

## 代码

```java
// leetcode submit region begin(Prohibit modification and deletion)
class Solution {
    private int quickSelect(List<Integer> nums, int k) {
        Random rand = new Random();
        int pivot = nums.get(rand.nextInt(nums.size()));
        List<Integer> big = new ArrayList<>();
        List<Integer> equal = new ArrayList<>();
        List<Integer> small = new ArrayList<>();

        for (int num : nums) {
            if (num > pivot) {
                big.add(num);
            } else if (num < pivot) {
                small.add(num);
            } else {
                equal.add(num);
            }
        }
        if (k <= big.size()) {
            return quickSelect(big, k);
        }
        if (nums.size() - small.size() < k) {
            return quickSelect(small, k - nums.size() + small.size());
        }
        return pivot;
    }

    public int findKthLargest(int[] nums, int k) {
        List<Integer> numList = new ArrayList<>();
        for (int num : nums) {
            numList.add(num);
        }
        return quickSelect(numList, k);
    }
}
```

## 复杂度

- Time: O(n) 平均，O(n²) 最坏（随机化 pivot 可避免）
- Space: O(n)（创建新列表），可优化到 O(1) 原地分区
