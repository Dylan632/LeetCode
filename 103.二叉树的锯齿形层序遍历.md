## 思路（必填）
### 问题分析
  - 要求按层遍历二叉树，但相邻层遍历方向相反
  - 第0层（根节点层）：从左到右
  - 第1层：从右到左
  - 第2层：从左到右
  - 依此类推...
### 算法选择
BFS层序遍历 + 方向控制
  - 使用BFS保证按层处理
  - 通过判断层号的奇偶性决定是否反转
### 实现步骤
  1. 特殊情况处理：根节点为空返回空列表
  2. 初始化：当前层为根节点列表
  3. 循环处理每一层：
    - 创建下一层节点列表
    - 遍历当前层所有节点：
        - 记录节点值
      - 收集子节点到下一层列表
    - 如果是奇数层，反转当前层结果
    - 将当前层结果加入答案列表
    - 更新当前层为下一层
  4. 返回最终结果

## 关键点
>[!Tip]
>- 使用 `ans.size()` 判断当前层号（第 0、1、2... 层）
>- 奇数层反转，偶数层保持
>- 子节点收集始终从左到右，只在结果层面反转

## 代码（两个数组）

```java
class Solution {
    public List<List<Integer>> zigzagLevelOrder(TreeNode root) {
        if (root == null) {
            return List.of();
        }
        List<List<Integer>> ans = new ArrayList<>();
        List<TreeNode> cur = List.of(root);
        while (!cur.isEmpty()) {
            List<TreeNode> nxt = new ArrayList<>();
            List<Integer> vals = new ArrayList<>(cur.size());
            for (TreeNode node : cur) {
                vals.add(node.val);
                if (node.left != null) {
                    nxt.add(node.left);
                }
                if (node.right != null) {
                    nxt.add(node.right);
                }
            }  
            cur = nxt;
            if (ans.size() % 2 > 0) {
                Collections.reverse(vals);
            }
            ans.add(vals);
        }
        return ans;
    }
}
```

## 代码（一个队列）
```java
class Solution {
    public List<List<Integer>> zigzagLevelOrder(TreeNode root) {
        if (root == null) {
            return List.of();
        }
        List<List<Integer>> ans = new ArrayList<>();
        Queue<TreeNode> q = new ArrayDeque<>();
        q.add(root);
        while (!q.isEmpty()) {
            int n = q.size();
            List<Integer> vals = new ArrayList<>(n);
            while (n-- > 0) {
                TreeNode node = q.poll();
                vals.add(node.val);
                if (node.left != null) {
                    q.add(node.left);
                }
                if (node.right != null) {
                    q.add(node.right);
                }
            }
            if (ans.size() % 2 > 0) {
                Collections.reverse(vals);
            }
            ans.add(vals);
        }
        return ans;
    }
}
```

### 📌 总结

| 特性 | 说明 |
|------|------|
| 是否原地修改 | ✅ 是，直接修改原 List |
| 时间复杂度 | O(n) |
| 空间复杂度 | O(1) |
| 适用类型 | 只支持 List（不支持 Set、Queue 等） |
| 不可变列表 | ❌ 会抛出 UnsupportedOperationException |
| 最佳实践 | ArrayList 性能最优，LinkedList 稍慢 |

**核心记忆点：**
- 原地反转，不创建新集合
- 使用双指针交换元素
- 时间 O(n)，空间 O(1)
- 只适用于可变 List
