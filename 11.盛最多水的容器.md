## 题目
![[11.png]]

## 思路（必填）

1. `left` 和 `right` 分别指向数组两端。
2. 循环计算当前面积：`min(height[left], height[right]) * (right - left)`。
3. 比较并保留最大面积。
4. 关键点：移动较短的那一侧指针。因为移动长板只会让容器的“天花板”保持不变或更低，而宽度还在减小，面积绝对不会变大；只有移动短板，才有可能遇到更高的新短板，从而补偿宽度的损失。

## 关键点
> [!Tip] 双指针技巧，移动较低的一边

## 代码

```java
class Solution {  
    public int maxArea(int[] height) {  
        int left = 0;  
        int right = height.length - 1;  
        int res = 0;  
  
        while (left < right) {  
            // 计算当前左右指针围成的水槽面积  
            // 面积 = 较短边的长度 * 左右指针的距离  
            int curArea = Math.min(height[left], height[right]) * (right - left);  
            res = Math.max(res, curArea);  
            // 贪心策略：移动较短的那根板  
            // 因为宽度在减小，只有寻找更高的板才有可能让面积增加  
            if (height[left] < height[right]) {  
                left++;  
            } else {  
                right--;  
            }  
        }  
        return res;  
    }  
}
```

## 知识点
双指针法利用了贪心思想：初始时宽度最大。当我们将指针向内移动时，宽度必然减小。为了使面积可能增大，我们必须让高度增加。

## 复杂度

- Time: O(n)
- Space: O(1)
