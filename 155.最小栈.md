## 思路（必填）

### 问题分析

- 设计一个支持 push、pop、top 和 getMin 操作的栈
- 要求所有操作的时间复杂度均为 O(1)
- 需要在常数时间内获取栈中的最小值

### 算法选择

选择辅助变量法（同步栈），将每个元素和其对应时刻的最小值一起存储在栈中。

### 实现步骤

1. 初始化哨兵：栈底放置一个哨兵节点 `[0, Integer.MAX_VALUE]`，避免空栈判断
2. 入栈操作：每次 `push(val)` 时，计算当前最小值 `min(栈顶最小值, val)`，将 `[val, min]` 一起入栈
3. 出栈操作：直接弹出栈顶元素，最小值信息自动更新到新的栈顶
4. 查询操作：`top()` 返回栈顶第一维，`getMin()` 返回栈顶第二维

## 关键点

> [!Tip] 核心技巧
> 1. **用栈模拟数组的「前缀最小值」**：栈中同时存储元素值和当前最小值
> 2. **哨兵简化边界**：栈底放 `Integer.MAX_VALUE`，避免判空逻辑

## 代码

```java
class MinStack {  
    // 注意不要使用 Stack 类，因为它继承自 Vector，是同步的，会导致一些性能问题  
    private final Deque<int[]> st = new ArrayDeque<>();  
  
    public MinStack() {  
        // 添加栈底哨兵 Integer.MAX_VALUE        
        // 这里的 0 写成任意数都可以，反正用不到  
        st.push(new int[]{0, Integer.MAX_VALUE});  
    }  
  
    public void push(int val) {  
        st.push(new int[]{val, Math.min(getMin(), val)});  
    }  
  
    public void pop() {  
        st.pop();  
    }  
  
    public int top() {  
        return st.peek()[0];  
    }  
  
    public int getMin() {  
        return st.peek()[1];  
    }  
}
```

## 知识点

### 核心思想：前缀最小值

#### 概念引入

给你一个数组 nums，如何计算每个前缀的最小值？

定义 `preMin[i]` 表示 `nums[0]` 到 `nums[i]` 的最小值。

**递推公式**：
- `preMin[0] = nums[0]`
- `preMin[i] = min(preMin[i-1], nums[i])`

#### 应用到栈

把 nums 视作栈，本题相当于在 nums 的末尾动态地添加/删除元素。

**栈中保存 pair**：`(当前元素, 当前前缀最小值)`

| 操作 | 逻辑 |
|------|------|
| 添加元素 | `preMin[n] = min(preMin[n-1], val)` |
| 删除元素 | 弹出栈顶即可 |

#### 细节：哨兵优化

**问题**：栈为空时，没有 `preMin[n-1]`

**解决**：栈底添加 `∞` 哨兵作为 `preMin[-1]`

> [!Note] 题目保证 `pop`、`top`、`getMin` 都是在非空栈上操作

## 复杂度

- Time: O(1) - 每个操作都是常数时间
- Space: O(n) - 栈存储所有元素
