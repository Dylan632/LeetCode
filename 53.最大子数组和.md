## 题目
![[53.png]]

## 思路（必填）
### 算法选择
动态规划，题目是"连续子数组最大和"，每个位置只需要"以当前位置结尾的最大和"，不需要回看全部历史。

### 实现步骤
1. 定义 currentSum：以当前元素结尾的最大子数组和。
2. 遍历每个 x，转移：currentSum = max(x, currentSum + x)。
3. 用 maxSum 维护全局最大值：maxSum = max(maxSum, currentSum)。
4. 遍历结束返回 maxSum。

## 关键点

>[!Tip] 核心要点
> 1. currentSum 的含义必须明确：它是"以当前元素结尾"的最大和，不是全局最大和。
> 2. 转移式核心二选一：max(x, currentSum + x)，决定"重开一段"还是"继续累计"。
> 3. maxSum 每轮都更新，负责记录全局最优，避免只看最后一段。
> 4. 初始化要能覆盖全负数组：maxSum = Integer.MIN_VALUE（或都初始化为 nums[0]）。

## 代码

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int currentSum = 0;
        int maxSum = Integer.MIN_VALUE;
        for (int x : nums) {
            currentSum = Math.max(currentSum + x, x);
            maxSum = Math.max(maxSum, currentSum);
        }
        return maxSum;
    }
}
```

## 知识点

1. 状态定义：局部最优（以 i 结尾）到全局最优。
2. 状态转移：要么"接上之前子数组"，要么"从当前重新开始"。
3. 边界处理：全负数组也能正确返回最大（最不负）元素。

## 复杂度

- Time: O(n)
- Space: O(1)
