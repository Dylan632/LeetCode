## 代码

```java
class Solution {  
    public ListNode mergeTwoLists(ListNode list1, ListNode list2) {  
        ListNode dummy = new ListNode();  
        ListNode cur = dummy;  
        while (list1 != null && list2 != null) {  
            if (list1.val > list2.val) {  
                cur.next = list2;  
                list2 = list2.next;  
            } else {  
                cur.next = list1;  
                list1 = list1.next;  
            }  
            //cur节点前移一位  
            cur = cur.next;  
        }  
        cur.next = (list1 != null) ? list1 : list2;  
        return dummy.next;  
    }  
}
```
