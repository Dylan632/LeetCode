## 题目
![[203.png]]

## 思路（必填）

1. 创建 dummy -> head，用 prev 指向 dummy。
2. 循环检查 prev.next 是否存在。
3. 若 prev.next.val == val，执行删除：prev.next = prev.next.next。
4. 否则 prev = prev.next 正常前进。
5. 返回 dummy.next 作为新头结点。

## 关键点
删除分支里不要移动 prev，否则会跳过连续待删节点。

## 代码

```java
class Solution {
    public ListNode removeElements(ListNode head, int val) {
        //创建虚拟头结点
        ListNode dummy = new ListNode();
        dummy.next = head;
        ListNode prev = dummy;
        while (prev.next != null) {
            if (prev.next.val == val) {
                prev.next = prev.next.next;
            } else {
                prev = prev.next;
            }
        }
        return dummy.next;
    }
}
```
