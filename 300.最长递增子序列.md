## 思路（必填）
### 算法选择

动态规划是解决最长递增子序列的经典方法。核心在于定义 **dp[i] 为以 nums[i] 结尾的最长递增子序列长度**，这样每个子问题只需要向前看比自己小的元素。

### 实现步骤
1. 初始化 dp 数组全为 1（每个元素自身构成长度 1 的子序列）
2. 对每个位置 i，遍历所有 j < i：若 nums[j] < nums[i]，说明 nums[i] 可以接在 nums[j] 后面，更新 dp[i] = max(dp[i], dp[j] + 1)
3. 最终答案是 dp 数组中的最大值（LIS 不一定以最后一个元素结尾）

## 关键点
> [!Tip] 状态定义必须是“以 nums[i] 结尾”，而不是“前 i 个元素中的 LIS”，否则状态转移无法正确进行。答案是 max(dp[0..n-1])，而非 dp[n-1]。

## 代码

```java
class Solution {  
    public int lengthOfLIS(int[] nums) {  
        // dp[i] 表示以 nums[i] 这个数结尾的最长递增子序列的长度  
        int[] dp = new int[nums.length];  
        // base case：dp 数组全都初始化为 1
        Arrays.fill(dp, 1);
        for (int i = 0; i < nums.length; i++) {  
            for (int j = 0; j < i; j++) {  
                if (nums[i] > nums[j]) {  
                    dp[i] = Math.max(dp[i], dp[j] + 1);  
                }  
            }  
        }  
        int res = 0;  
        for (int i = 0; i < dp.length; i++) {  
            res = Math.max(res, dp[i]);  
        }  
        return res;  
    }  
}
```

## 优化代码：贪心 + 二分查找（O(n log n)）

```java
class Solution {
    public int lengthOfLIS(int[] nums) {
        if (nums == null || nums.length == 0) return 0;

        // tails[i] 表示长度为 i+1 的递增子序列的最小末尾元素
        int[] tails = new int[nums.length];
        int size = 0; // 当前最长递增子序列的长度

        for (int num : nums) {
            // 二分查找：在 tails[0..size) 中找第一个 >= num 的位置
            int low = 0, high = size;
            while (low < high) {
                int mid = low + (high - low) / 2;
                if (tails[mid] < num) {
                    low = mid + 1;
                } else {
                    high = mid;
                }
            }
            tails[low] = num;
            if (low == size) size++; // num 比所有末尾都大，LIS 长度 +1
        }
        return size;
    }
}
```

## 复杂度

### 动态规划

- Time：O(n²)
- Space：O(n)

### 贪心 + 二分查找

- Time：O(n log n)
- Space：O(n)
