## 题目
![[148.png]]

## 思路

使用**自底向上的归并排序**，通过迭代方式实现 O(1) 空间复杂度。

### 算法选择

归并排序（从底至顶直接合并）

### 核心思想

非递归的归并排序通过迭代替换递归的 cut 环节：
- 每一轮合并针对固定长度 `step` 的单元
- `step = 1`：将链表切分为多个长度为 1 的单元，两两合并
- `step = 2`：将链表切分为多个长度为 2 的单元，两两合并
- 以此类推，直到 `step >= 链表长度`，排序完成

### 实现步骤

![[148_1.png]]

1. **统计链表长度** `length`，用于判断是否完成排序
2. **声明 dummy 节点**，作为辅助头部，便于：
   - 通过 `dummy.next` 找到链表头部
   - 作为排序合并时的辅助节点
3. **每轮 step 下的合并流程**：
   - 从 `cur` 开始，分割出两段长为 `step` 的链表，头节点分别为 `head1` 和 `head2`
   - 合并两段链表
   - 将合并后的链表连接到新链表末尾
   - 更新 `cur` 为下一轮循环的起始节点
4. **更新步长**：每轮合并完成后 `step *= 2`

## 关键点

> [!Tip] 归并排序的核心是”分而治之”
> - **分割**：将链表切成若干长度为 `step` 的子链表
> - **合并**：将相邻的两个有序子链表合并成一个更大的有序链表
> - **迭代**：逐步增大 `step`，直到整个链表有序

## 代码

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode sortList(ListNode head) {
        int length = getListLength(head);
        ListNode dummy = new ListNode(0, head);
        // step 为步长，即参与合并的链表长度
        for (int step = 1; step < length; step *= 2) {
            // 新链表的末尾
            ListNode newListTail = dummy;
            // 每轮循环的起始节点
            ListNode cur = dummy.next;
            while (cur != null) {
                // 从 cur 开始，分割出两段长为 step 的链表，头节点分别为 head1 和 head2
                ListNode head1 = cur;
                ListNode head2 = splitList(head1, step);
                // 下一轮循环的起始节点
                cur = splitList(head2, step);
                // 合并两段长为 step 的链表
                ListNode[] merged = mergeTwoLists(head1, head2);
                // 合并后的头节点 merged[0]，插到 newListTail 的后面
                newListTail.next = merged[0];
                // merged[1] 现在是新链表的末尾
                newListTail = merged[1];
            }
        }
        return dummy.next;
    }

    // 获取链表长度
    private int getListLength(ListNode head) {
        int length = 0;
        while (head != null) {
            length++;
            head = head.next;
        }
        return length;
    }

    // 分割链表
    // 如果链表长度 <= size，不做任何操作，返回空节点
    // 如果链表长度 > size，把链表的前 size 个节点分割出来（断开连接），并返回剩余链表的头节点
    private ListNode splitList(ListNode head, int size) {
        // 先找到 nextHead 的前一个节点
        ListNode cur = head;
        for (int i = 0; i < size - 1 && cur != null; i++) {
            cur = cur.next;
        }
        // 如果链表长度 <= size
        if (cur == null || cur.next == null) {
            return null;
        }
        ListNode nextHead = cur.next;
        // 断开 nextHead 的前一个节点和 nextHead 的连接
        cur.next = null;
        return nextHead;
    }

    // 返回合并后的链表的头节点和尾节点
    private ListNode[] mergeTwoLists(ListNode list1, ListNode list2) {
        ListNode dummy = new ListNode();
        // cur 指向新链表的末尾
        ListNode cur = dummy;
        while (list1 != null && list2 != null) {
            if (list1.val < list2.val) {
                // 把 list1 加到新链表中
                cur.next = list1;
                // 将 list1 指针向后移动一位，指向下一个节点
                list1 = list1.next;
            } else {
                cur.next = list2;
                list2 = list2.next;
            }
            cur = cur.next;
        }
        // 拼接剩余链表
        cur.next = list1 != null ? list1 : list2;
        while (cur.next != null) {
            cur = cur.next;
        }
        // 循环结束后，cur 是合并后的链表的尾节点
        return new ListNode[]{dummy.next, cur};
    }
}
```

## 知识点

### 自底向上归并排序

非递归归并排序通过迭代替换递归的 cut 环节：

1. **cut 环节本质**：通过二分法得到链表最小节点单元，再通过多轮合并得到排序结果
2. **每轮合并**：针对固定长度 `step` 的单元进行两两合并

| step | 操作 | 合并后单元长度 |
|------|------|----------------|
| 1 | 第 1、2 节点合并，第 3、4 节点合并，…… | 2 |
| 2 | 第 1-2 和第 3-4 合并，第 5-6 和第 7-8 合并，…… | 4 |
| 4 | 第 1-4 和第 5-8 合并，第 9-12 和第 13-16 合并，…… | 8 |
| ... | 直到 step >= 链表长度 | ... |

### 辅助方法

| 方法 | 功能 |
|------|------|
| `getListLength` | 统计链表长度 |
| `splitList` | 分割链表，返回剩余部分的头节点 |
| `mergeTwoLists` | 合并两个有序链表，返回头节点和尾节点 |

> [!Note] 为什么返回头节点和尾节点？
> 合并后需要将结果连接到新链表末尾，返回尾节点可以避免再次遍历找尾部，提高效率。

## 复杂度

- **Time**: O(n log n)，n 是链表长度
- **Space**: O(1)，只使用常数级别的额外空间
