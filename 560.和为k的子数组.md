## 思路

### 问题分析

- 题目要求：找到和为 k 的连续子数组的个数
- 难点：子数组可以是任意长度，暴力枚举需要 $O(n^2)$ 时间

### 算法选择

当我们遍历到当前前缀和 `prefixSum` 时，只要看看之前有多少个前缀和等于 `prefixSum - k`，就说明有多少个子数组和为 `k`。

- 前缀和 + 哈希表：将二维问题降维为一维
- 数学原理：$\text{sum}(i, j) = \text{prefixSum}[j] - \text{prefixSum}[i-1] = k$
- 等价转化：查找 $\text{prefixSum}[i-1] = \text{prefixSum}[j] - k$ 的出现次数

### 实现步骤

1. 初始化哈希表 `{0: 1}`（前缀和 0 出现 1 次，代表空前缀）
2. 遍历数组，累加计算当前位置的前缀和 `prefixSum`
3. 查找哈希表中 `prefixSum - k` 的出现次数，累加到答案
4. 将当前 `prefixSum` 加入哈希表，更新其出现次数

## 关键点

> [!Tip] 子数组的和，就等于**两个前缀和的差**。

- ✅ 先查找再更新：避免子数组长度为 0 的错误（当 k=0 时）
- ✅ 初始哈希表包含 `{0: 1}`：正确处理从数组开头到当前位置的子数组

## 代码

```java
class Solution {
    public int subarraySum(int[] nums, int k) {
        // key: 前缀和, value: 该前缀和出现的次数
        Map<Integer, Integer> cnt = new HashMap<>(nums.length + 1, 1);
        cnt.put(0, 1); // 默认前缀和为 0 的子数组出现 1 次
        int prefixSum = 0;
        int ans = 0;
        for (int x : nums) {
            prefixSum += x;
            ans += cnt.getOrDefault(prefixSum - k, 0);
            cnt.merge(prefixSum, 1, Integer::sum);
        }
        return ans;
    }
}
```

## 知识点

### Java 8 语法糖 `merge`

```java
cnt.merge(prefixSum, 1, Integer::sum);
```

一句话搞定“没有就存 1，有就加 1”。

**`merge` 方法原理：**

```java
// 等价于
if (cnt.containsKey(prefixSum)) {
    cnt.put(prefixSum, cnt.get(prefixSum) + 1);
} else {
    cnt.put(prefixSum, 1);
}
```

**`Integer::sum` 是什么？**

```java
public static int sum(int a, int b) {
    return a + b;  // 就是简单的加法
}
```

`::` 是方法引用语法，`Integer::sum` 相当于把加法操作作为参数传入。当键已存在时，`merge` 会用这个函数合并旧值和新值。
