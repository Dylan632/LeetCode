## 思路（必填）
### 算法选择

### 实现步骤
1. 栈底先放 -1，作为初始哨兵。
2. 遇到 `(`，把下标入栈。
3. 遇到 `)`：
    - 如果栈里有可匹配的 `(`，弹出一个，然后用 `i - 栈顶`，更新当前最长长度。
    - 如果没有可匹配的 `(`，把当前下标设为新的哨兵。
4. 遍历结束后返回最大长度

## 关键点
>[!Tip] 栈里存的是“下标”，不是字符，因为最后要算长度；弹栈之后新的栈顶，代表当前有效括号段左边界之前的位置；哨兵的作用是统一处理开头就遇到 ')' 或中间断开的情况。

## 代码

```java
class Solution {  
    public int longestValidParentheses(String s) {  
        List<Integer> stk = new ArrayList<>();  
        stk.add(-1); // 虚拟红线（哨兵）  
        int ans = 0;  
        for (int i = 0; i < s.length(); i++) {  
            if (s.charAt(i) == '(') { // 炸弹  
                stk.add(i); // 记录炸弹下标  
            } else if (stk.size() > 1) { // 栈顶是炸弹  
                stk.removeLast(); // 拆弹  
                // 右端点为 i 时，左端点最小是 stk.getLast() + 1                
                ans = Math.max(ans, i - stk.getLast());  
            } else {  // 栈只有一个数，是红线，s[i] 无法拆弹，成为新的红线  
                stk.set(0,i); // 替换栈底  
            }  
        }  
        return ans;  
    }  
}
```
## 举例实现步骤
假设字符串是 `s = "(()"`
- **初始**：栈为 `[-1]` （虚拟红线）
- `i = 0`，遇到 `(` ：入栈，栈变成 `[-1, 0]`
- `i = 1`，遇到 `(` ：入栈，栈变成 `[-1, 0, 1]`
- `i = 2`，遇到 `)` ：此时发生配对！
    1. **拆弹**：先把 `i=1` 的那个左括号出栈，栈变回 `[-1, 0]`。
    2. **找边界**：现在的栈顶是 `0`（代表第 0 个位置的那个 `(` 还没有被匹配掉）。
    3. **算端点**：对于当前的右端点 `i=2`，阻碍它往左延伸的边界是下标 `0`。所以，当前的有效子串只能从 `0 + 1 = 1` 开始。
    4. **计算长度**：这段有效子串（即中间的 `()`）的长度是 `2 - 0 = 2`。

## 怎么理解“右端点为 i 时，左端点最小是 stk.getLast() + 1” 

要理解这句话，我们需要搞清楚在这行代码执行时，**现在的栈顶元素 `stk.getLast()` 到底代表着什么**。
简单直接地说：在你刚刚弹出一个左括号（完成一次“拆弹”配对）之后，**当前栈顶剩下的那个元素，就是阻碍你继续向左延伸的“边界”或“路障”。**
下面为您详细拆解其中的逻辑：
### 1. `stk.getLast()` 究竟是什么？
在执行了 `stk.removeLast()` 之后，当前的栈顶元素 `stk.getLast()` 记录的是**最后一个尚未被匹配的字符的下标**（或者是一开始设置的虚拟红线 `-1`）。
这个位置可能是：
- **初始的 `-1`**：说明前面所有的括号都已经完美匹配了，没有任何多余的右括号阻挡你。
- **一个多余的右括号 `)` 的下标**：说明在那个位置发生了断层，导致连续性被破坏（对应代码里的 `stk.set(0,i)` 产生的新红线）。
- **一个还没被匹配的左括号 `(` 的下标**：说明外面还有包裹着你的左括号没被闭合（例如 `(( )`，匹配了里面那一对，外面那个左括号就是边界）。
### 2. 为什么左端点是 `stk.getLast() + 1`？
既然 `stk.getLast()` 代表的是“不可跨越的边界”或“未匹配的坏字符”，那么你当前刚刚配对成功的这一长串**有效的连续括号，必然紧紧贴在这个边界的右侧**。
所以，有效子串真正的起始位置（左端点），就是边界的下一位，即：`stk.getLast() + 1`。
### 3. 为什么说是“最小”？
我们要找的是**最长**有效括号。要让长度最长，右端点固定在当前的 `i` 时，左端点就必须尽可能靠左，也就是**最小**。 因为你最多只能向左延伸到那个“未匹配边界”的后面一位，再往左就跨越了无效字符了，所以 `stk.getLast() + 1` 就是你能争取到的最极限、最小的合法左端点。
有了这段合法子串的范围 `[stk.getLast() + 1, i]`，它的长度计算公式就是： `右端点 - 左端点 + 1 = i - (stk.getLast() + 1) + 1 = i - stk.getLast()` 这就是代码中直接用 `i - stk.getLast()` 来更新答案的原因。
