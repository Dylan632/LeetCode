## 思路（必填）

### 问题分析
要求按层遍历二叉树，将同一层的节点值放在同一个列表中，最终返回列表的列表。

### 算法选择：BFS 广度优先搜索

- BFS 天然适合层序遍历，保证先访问上层节点再访问下层节点
- 使用队列实现 FIFO，保证节点按层级顺序处理

### 实现步骤

1. 特殊情况处理：根节点为空时返回空列表
2. 初始化队列，将根节点入队
3. 循环处理每一层：
   - 记录当前层节点数 n
   - 依次处理 n 个节点：出队、记录值、子节点入队
   - 将当前层结果加入答案列表
4. 返回最终结果

## 💭 代码思路（详细讲解）

```
        3
       / \
      9   20
         /  \
        15   7
```

目标输出：`[[3], [9, 20], [15, 7]]`——按层分组。

**核心思想：用队列模拟“排队”。**

想象一个售票窗口，树的节点是来买票的人。规则是：
- 每个人买完票后，把自己的孩子叫来排队
- 我们需要记录“第几批人一起在排队”

---

### 逐行执行过程

1. 创建空的结果列表 `ans = []`，创建空队列 `q = []`
2. 根节点 3 入队 → `q = [3]`
3. 进入 while 循环（队列非空）

**第一轮 while（处理第 1 层）**：
`q = [3]`，`n = 1`（当前层有 1 个节点）
while 循环执行 1 次（`n-- > 0`）：
- 取出 3，`vals = [3]`，把 3 的左孩子 9 和右孩子 20 入队
`q = [9, 20]`，`ans = [[3]]`

**第二轮 while（处理第 2 层）**：
`q = [9, 20]`，`n = 2`（当前层有 2 个节点）
while 循环执行 2 次（`n-- > 0`）：
- 第 1 次：取出 9，`vals = [9]`，9 没有孩子，不入队
- 第 2 次：取出 20，`vals = [9, 20]`，把 20 的左孩子 15 和右孩子 7 入队
`q = [15, 7]`，`ans = [[3], [9, 20]]`

**第三轮 while（处理第 3 层）**：
`q = [15, 7]`，`n = 2`
while 循环执行 2 次（`n-- > 0`）：
- 第 1 次：取出 15，`vals = [15]`，无孩子
- 第 2 次：取出 7，`vals = [15, 7]`，无孩子
`q = []`（空了），`ans = [[3], [9, 20], [15, 7]]`

队列为空，while 循环结束，返回 ans。

整个过程的核心就两件事：
1. 出队一个节点 → 记录它的值
2. 入队它的孩子 → 留给下一层处理

`n` 就像一个“闸门”，确保每轮只处理当前层的节点。

## 关键点

> [!Tip]
> `n = q.size()` 必须在内层循环之前记录，因为循环过程中会往队列里添加下一层的节点，如果不提前记录，就无法区分当前层和下一层的边界。

## 代码（一个队列）

```java
class Solution {
    public List<List<Integer>> levelOrder(TreeNode root) {
        if (root == null) {
            return List.of();
        }
        List<List<Integer>> ans = new ArrayList<>();
        Queue<TreeNode> q = new ArrayDeque<>();
        q.add(root);
        while (!q.isEmpty()) {
            int n = q.size();
            List<Integer> vals = new ArrayList<>(n); // 预分配空间
            while (n-- > 0) {
                TreeNode node = q.poll();
                vals.add(node.val);
                if (node.left != null)  q.add(node.left);
                if (node.right != null) q.add(node.right);
            }
            ans.add(vals);
        }
        return ans;
    }
}
```
## 灵神代码（两个数组）

```java
class Solution {  
    public List<List<Integer>> levelOrder(TreeNode root) {  
        if (root == null) {  
	        // 返回一个不可变的空的二维列表
            return List.of();  
        }  
        List<List<Integer>> ans = new ArrayList<>();  
        List<TreeNode> cur = List.of(root);  
        while (!cur.isEmpty()) {  
            List<TreeNode> nxt = new ArrayList<>();  
            List<Integer> vals = new ArrayList<>(cur.size());  
            for (TreeNode node : cur) {  
                vals.add(node.val);  
                if (node.left != null) {  
                    nxt.add(node.left);  
                }  
                if (node.right != null) {  
                    nxt.add(node.right);  
                }  
            }  
            cur = nxt;  
            ans.add(vals);  
        }  
        return ans;  
    }  
}
```

### 📝 动态运行推演（以一个简单二叉树为例）

假设我们有一棵这样的树：

```text
    1
   / \
  2   3
 /
4
```

- 初始状态：`cur = [Node(1)]`
- **第一轮 while 循环**：
  - `nxt` 准备为空，`vals` 准备为空
  - 遍历 `cur`（只有节点 1）：记录值 `vals = [1]`。节点 1 有左孩子 2 和右孩子 3，放入下一层：`nxt = [Node(2), Node(3)]`
  - 滚动切换：`cur = [Node(2), Node(3)]`，`ans = [[1]]`

- **第二轮 while 循环**：
  - 遍历 `cur`（节点 2 和节点 3）：记录值 `vals = [2, 3]`。节点 2 有左孩子 4，放入 `nxt`。节点 3 没有孩子。所以 `nxt = [Node(4)]`
  - 滚动切换：`cur = [Node(4)]`，`ans = [[1], [2, 3]]`

- **第三轮 while 循环**：
  - 遍历 `cur`（只有节点 4）：记录值 `vals = [4]`。节点 4 没有孩子，`nxt` 保持为空 `[]`
  - 滚动切换：`cur = []`，`ans = [[1], [2, 3], [4]]`
- **第四轮 while 循环**：
  - `cur` 为空，跳出循环
- 最终返回 `ans`

## 知识点

### 1. ans 的数据类型

```java
List<List<Integer>> ans = new ArrayList<>();
```

- `ArrayList<>`：动态数组，是 `List` 接口的一种具体实现，底层用数组存数据，支持自动扩容
- 数据类型：`List<List<Integer>>`
- 含义：
  - `List<...>`：一个“列表/动态数组”
  - 里面放的是 `List<Integer>`，也就是“一层的所有节点值”
- 所以：`ans` 是一个“列表的列表”：
  - `ans.get(0)`：第 0 层（根节点这一层）的所有值列表
  - `ans.get(1)`：第 1 层的所有值列表
  - ...
- 结构大概是这样：

```
[
  [第0层的所有值],
  [第1层的所有值],
  [第2层的所有值],
  ...
]
```

### 2. vals 的数据类型

```java
List<Integer> vals = new ArrayList<>(n);
```

- 数据类型：`List<Integer>`
- 含义：
  - `vals` 表示“当前这一层的所有节点的值”的列表
  - 在 `while (!q.isEmpty())` 的每一轮循环里：
    - 新建一个 `vals`，只存当前这一层的节点的值
    - 一层处理完之后，执行 `ans.add(vals);`，把这一层的结果存进总结果 `ans` 里
- 可以理解为：
  - `ans` 是“总成绩单（很多行）”
  - `vals` 是“这一行的成绩”

### 3. node 的数据类型

```java
TreeNode node = q.poll();
```

- 数据类型：`TreeNode`
- 含义：`TreeNode` 是题目定义的“二叉树节点”类：

```java
public class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
}
```

- 所以 `node` 是“当前正在处理的这个树节点”
- 你可以通过：
  - `node.val` 访问当前节点的值（整数）
  - `node.left` 访问它的左孩子
  - `node.right` 访问它的右孩子

### 4. node.val

- `node`：指向当前二叉树结点的变量（通常类型是 `TreeNode`）
- `val`：这个结点里存储的数值字段，在 `TreeNode` 类里一般定义为 `int val;`

所以 `vals.add(node.val);` 就是：把当前这个结点中存的数值（`val`）加入到当前这一层的列表 `vals` 里。

## 复杂度

- Time: O(n)
- Space: O(n)

每个节点入队出队各一次；队列最多存一层节点（最坏 n/2）。
