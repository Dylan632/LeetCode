## 代码

```java
class Solution {  
    //一维列表  
    public List<Integer> resList = new ArrayList<Integer>();  
    public List<Integer> inorderTraversal(TreeNode root) {  
        checkFun(root);  
        return resList;  
    }  
    public void checkFun(TreeNode node) {  
        if (node == null) {  
            return;  
        }  
        checkFun(node.left);  
        //中序遍历，add在中间
        resList.add(node.val);  
        checkFun(node.right);  
    }  
}
```
