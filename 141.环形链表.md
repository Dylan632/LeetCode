## 题目
![[141.png]]

## 思路（必填）

1. 处理空链表/单节点边界。
2. 初始化 slow、fast 都指向头节点。
3. 循环推进：slow +1，fast +2。
4. 若相遇返回 true；循环结束返回 false。

## 关键点
fast每次移动两位，要确保fast及其下一位不是null。
循环条件必须写成 fast != null && fast.next != null，否则 fast.next.next 有空指针风险。

## 代码

```java
public class Solution {  
    public boolean hasCycle(ListNode head) {  
        ListNode fast = head, slow = head;  
        while (fast != null && fast.next != null) {  
            slow = slow.next;  
            fast = fast.next.next;  
            if (slow == fast) {  
                return true;  
            }  
        }  
        return false;  
    }  
}
```
