## 关键点
fast每次移动两位，要确保fast及其下一位不是null。

## 代码

```java
public class Solution {  
    public boolean hasCycle(ListNode head) {  
        ListNode fast = head, slow = head;  
        while (fast != null && fast.next != null) {  
            slow = slow.next;  
            fast = fast.next.next;  
            if (slow == fast) {  
                return true;  
            }  
        }  
        return false;  
    }  
}
```
