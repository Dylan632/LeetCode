## 第一章 绪论

### 研究背景与意义

以时间顺序列写国内外风光互补发电系统的研究历史、现状与发展趋势，并重点介绍南极部分科考站能源供电系统的发展历程，阐明以南极无人观测站为研究对象的工程意义与科学价值。

### 现有研究综述

**风光发电侧控制方向**

在风光发电侧控制方面，MPPT 已形成较成熟的研究基础。围绕光伏 MPPT，现有文献对扰动观察法、增量电导法、模糊控制法等典型方法做了较系统的总结\[4\]\[5\]。风能转换系统的控制方法也已有较充分的研究基础\[6\]。

**微网能量管理方向**

在微网能量管理方面，近几年相关研究明显增多。现有综述普遍认为，能量管理的核心在于协调分布式电源、储能和负载之间的关系\[7\]\[8\]。Mahjoub 等利用预测信息对 PV-风机-电池微网进行能量管理，并开展了实验验证\[9\]；Touré 等采用模型预测控制方法研究含储能微网的能量调度问题\[10\]；Galván 等将负载优先级引入储能调度，说明在供能受限时按负载重要程度分配功率具有可行性\[11\]。Li 等对风光储 AC/DC 微网的平滑切换和稳定性进行了研究，也说明储能协调在系统运行中处于关键位置\[12\]。

### 现有研究的不足与本课题切入点

总体来看，现有研究为本课题提供了较好的基础，但仍存在以下几个值得继续深入的方向：

1. 专门面向南极无人观测站这类**长期无人值守场景**的工程化研究还不多。
2. 不少成果更关注预测优化或经济调度，而本课题更关注在**现有设备条件下如何保障系统稳定运行**。
3. 围绕**关键研究仪器优先供电**的储能充放电与功率分配研究还不够充分。

这些问题正是本课题进一步展开研究的切入点。

## 第二章 风光储系统结构与原理

### 对应研究内容（1）：风光储混合供能系统运行模式与能量管理研究

结合南极无人观测站供能系统的拓扑结构，分析风力发电单元、光伏发电单元、储能单元、逆变单元、负载及能量管理系统之间的连接关系和运行特性，梳理系统能量流动路径，明确风能、光能经变换后进入储能侧，再经逆变单元向负载供电的基本过程。

在此基础上，研究风光发电侧输出波动特性、储能单元在不同荷电状态下的支撑能力，以及系统在以下典型工况下的运行模式：

| 典型工况 | 描述 |
|---------|------|
| 风光充足 | 风光出力满足负载需求并可向储能充电 |
| 风光不足 | 风光出力不足，储能放电补充 |
| 储能可支撑 | 储能 SOC 处于正常区间，可持续向负载供电 |
| 储能受限 | 储能 SOC 过低，需启动负载削减策略 |

### 对应研究方法（1）：系统分析与工况建模方法

结合现有工程平台，对风力发电单元、光伏发电单元、储能单元、逆变单元及负载设备进行结构分析和运行特性分析，梳理各单元之间的能量传递关系和控制关系。结合风速、光照、储能荷电状态和负载功率需求等因素，划分系统典型运行工况，建立风光储系统在不同条件下的运行模式，为后续储能控制和功率分配策略设计提供依据。

### 章节主要内容

- 风力发电系统结构与原理（风机特性、风能转换原理）
- 光伏发电系统结构与原理（PV 电池特性、DC-DC 变换器）
- 蓄电池储能系统结构与原理（充放电特性、SOC 估算方法）
- 系统整体拓扑与能量流动路径分析

## 第三章 风光储各子系统控制策略

### 对应研究内容（部分）：风光发电侧 MPPT 控制与储能充放电控制

本章涵盖第四章能量管理策略所依赖的底层控制基础，包括：

- **风机系统最大功率控制策略（MPPT）**：扰动观察法、最优转矩控制等
- **光伏系统最大功率控制策略（MPPT）**：扰动观察法、增量电导法、模糊控制法等
- **蓄电池储能系统控制策略**：恒流/恒压充电控制、SOC 管理

### 对应研究方法（2）：控制策略设计与实验验证方法（底层部分）

在分析风光出力特性、储能荷电状态和负载需求变化规律的基础上，建立储能充放电控制规则，设计风机、光伏和储能之间的功率分配机制，明确不同工况下系统的运行逻辑和调度方式。

## 第四章 风光储系统能量管理策略

### 对应研究内容（2）：面向关键负载保障的能量管理与调度策略研究

在系统运行机理分析的基础上，围绕储能充放电控制、风光储功率分配和关键负载优先供电开展研究。重点分析风光出力变化、储能荷电状态变化以及负载需求变化对系统运行的影响，研究不同工况下储能单元的充放电策略和功率分配方式，构建适用于南极无人观测站的能量管理与调度策略。

同时，根据负载重要程度建立关键负载与一般负载的优先级划分方法：

```
负载优先级划分：
  ├── 一级负载（关键负载）：研究仪器、数据采集设备、通信设备
  └── 二级负载（一般负载）：照明、加热等辅助设备
```

在能源受限条件下优先保障研究仪器等关键负载稳定运行，并兼顾系统整体供能的连续性和安全性。

### 对应研究方法（2）：控制策略设计与实验验证方法（能量调度部分）

依托实际平台开展联调与实验测试，通过不同工况下运行数据的采集、对比和分析，验证所提策略在以下方面的有效性：

- 供电稳定性
- 关键负载保障
- 能源利用效率

并据此对控制参数和运行规则进行优化。

## 第五章 风光储系统的设计与实现

### 对应研究内容（3）：基于现有 PLC 控制逻辑的 C 语言重构与嵌入式平台部署研究

针对传统 PLC 在执行复杂多目标能量管理优化算法时存在的算力和灵活性限制，本部分研究风光储系统现有 PLC 控制逻辑向 C 语言程序的等效重构方法。

**重构目标**

围绕原有 PLC 程序中的状态机逻辑、梯形图时序关系以及 I/O 映射机制，分析其在软件层面的实现过程，并将其转化为结构清晰、便于扩展的 C 语言代码。

**部署架构**

```
目标平台（基于 Linux 的工控机）
  ├── 数据采集层        ← 底层总线通信（CAN / RS485）
  ├── 能量调度算法层    ← 重构后的控制逻辑与能量管理策略
  └── 硬件驱动层        ← 串口读写、工业总线通信库
```

**可靠性机制**

引入看门狗和多线程异常捕获机制，提高控制程序在极地无人值守和复杂工况下的实时响应能力与运行可靠性。

### 对应研究内容（4）：多节点异构通信协同与实物平台联调验证

能量管理策略能否稳定执行，很大程度上取决于底层设备之间的数据交互是否及时、准确。结合南极供能系统设备多、接口异构的特点，本部分研究控制器与以下设备之间的实时通信协同机制：

- 储能 BMS
- 光伏和风机逆变单元
- 环境温度监测模块等分布式硬件

**通信网络架构**

围绕 CAN 总线、RS485 等工业现场总线，设计数据采集与指令下发网络架构，解决多源异构数据在解析、时间同步和通信时延方面的问题。

**联调验证方案**

依托现有风光储实物工程平台开展系统级联调，并设置极端突变工况下的测试用例，对以下内容进行整体验证：

| 验证对象 | 评估指标 |
|---------|---------|
| 能量管理策略 | 关键负载保障率、供电连续性 |
| 底层部署代码 | 响应时延、执行稳定性 |
| 通信架构 | 数据同步精度、通信可靠性 |

### 对应研究方法（3）：PLC 逻辑等效解析与 C 语言跨平台软件化方法

采用"自顶向下解析、底层模块重构"的软件工程方法开展控制系统跨平台迁移：

1. 对原有 PLC 梯形图和指令表进行反向分析，梳理关键保护逻辑与控制时序，并通过状态机模型实现等效表达。
2. 在 Linux C 开发环境下按照模块化思路重构控制程序，分别建立数据采集层、能量调度算法层和硬件驱动层。
3. 针对硬件交互部分，采用成熟的底层通信库完成串口读写和工业总线通信，替代原 PLC 体系下的物理 I/O 映射方式。
4. 通过交叉编译将重构后的 C 代码部署到目标硬件，并在程序中加入容错与冗余机制，降低强电磁干扰等因素引发程序异常或失控的风险。

### 对应研究方法（4）：异构总线通信协同与"等效工况注入"联调测试方法

在实物验证阶段，不直接依赖真实极端环境，而是采用**"等效工况注入"**的方式开展联调测试：

- 利用**可编程直流电源**模拟极地条件下波动变化的风光输入特性
- 利用**可编程电子负载**模拟观测站设备的突发用能需求
- 将前期提取的**南极典型波动数据**按时间序列注入实物平台

通过对系统关键节点电压、电流和功率分配响应的采集与对比，评估 C 语言部署代码的执行性能以及调度策略在实际系统中的有效性。
