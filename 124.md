## 代码

```java
class Solution {
    private int ans = Integer.MIN_VALUE;
    public int maxPathSum(TreeNode root) {
        dfs(root);
        return ans;
    }
    private int dfs(TreeNode node) {
        if (node == null) {
            // 没有节点，和为 0
            return 0;
        }
        // 左子树最大链和
        int lVal = dfs(node.left);
        // 右子树最大链和
        int rVal = dfs(node.right);
        // 两条链拼成路径
        ans = Math.max(ans, lVal + rVal + node.val);
        // 当前子树最大链和（注意这里和 0 取最大值了）
        return Math.max(Math.max(lVal, rVal) + node.val, 0);
    }
}
```

## 递归树执行图
结合 `root = [-10,9,20,null,null,15,7]`，走一遍“后序遍历 + 树形DP”的流程。
```
      -10
     /   \
    9     20
         /  \
        15   7
```

时刻记住 `dfs(node)` 的核心任务：

1. **计算当前节点的路径和（左链 + 右链 + 自己）**，并尝试更新全局的 `ans`。
2. **向父节点返回单边最大链（max(左链, 右链) + 自己）**，如果算出来是负数就返回 0（代表不选）。

```
dfs(-10)
│
├── dfs(9)
│   ├── dfs(null) → return 0
│   └── dfs(null) → return 0
│   → lVal=0, rVal=0
│   → ans = max(-∞, 0 + 0 + 9) = 9  ✨ 更新！
│   → return max(max(0,0)+9, 0) = 9
│
└── dfs(20)
    │
    ├── dfs(15)
    │   ├── dfs(null) → return 0
    │   └── dfs(null) → return 0
    │   → lVal=0, rVal=0
    │   → ans = max(9, 0 + 0 + 15) = 15  ✨ 更新！
    │   → return max(max(0,0)+15, 0) = 15
    │
    └── dfs(7)
        ├── dfs(null) → return 0
        └── dfs(null) → return 0
        → lVal=0, rVal=0
        → ans = max(15, 0 + 0 + 7) = 15  (不变)
        → return max(max(0,0)+7, 0) = 7
    │
    → lVal=15 (来自节点15), rVal=7 (来自节点7)
    → ans = max(15, 15 + 7 + 20) = 42  ✨ 更新！(15->20->7 这条路径)
    → return max(max(15,7)+20, 0) = 35
│
→ lVal=9 (来自节点9), rVal=35 (来自节点20)
→ ans = max(42, 9 + 35 + (-10)) = max(42, 34) = 42  (34没打破纪录，ans保持42)
→ return max(max(9,35)+(-10), 0) = 25

全局变量 ans 最终结果: 42
```



## 知识点
本题有两个关键概念：

- 链：从下面的某个节点（不一定是叶子）到当前节点的路径。把这条链的节点值之和，作为 dfs 的返回值。如果节点值之和是负数，则返回 0（和 0 取最大值）。这个思想和 [[53.最大子数组和]] 是一样的，如果左侧子数组的元素和是负数，就不和当前元素拼起来。
- 直径：等价于由两条（或者一条）链拼成的路径。我们枚举每个 node，假设直径在这里「拐弯」，也就是计算由左右两条从下面的某个节点（不一定是叶子）到 node 的链的节点值之和，去更新答案的最大值。

>[!Tip] 
>dfs 返回的是链的节点值之和，不是直径的节点值之和。

问：如果所有节点值都是负数，代码会算出什么结果？

答：在所有节点值都为负数的情况下，代码中的 ans = max(ans, l_val + r_val + node.val) 等价于 ans = max(ans, node.val)，我们求的是最大节点值（绝对值最小的负数）。这是符合题目要求的，因为在所有节点值都为负数的情况下，路径只有一个节点是最优的（毕竟节点越多，元素和越小）。类比 [[53.最大子数组和]]，当数组元素都是负数的时候，答案就是 max(nums)。

## 复杂度

- Time: O(n)，其中 n 为二叉树的节点个数。
- Space: O(n)。最坏情况下，二叉树退化成一条链，递归需要 O(n) 的栈空间。
