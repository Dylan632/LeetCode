## 思路（必填）
### 算法选择
位运算异或。
### 实现步骤
1. 初始化累积值为 0。
2. 遍历数组并逐个异或。
3. 成对元素互相抵消，最终剩下只出现一次的元素。

## 关键点
>[!Tip] a ^ a = 0，0 ^ x = x，异或满足交换律和结合律。

## 代码

```java
class Solution {  
    public int singleNumber(int[] nums) {  
        int res = 0;  
        for (int n : nums){  
            res ^= n;  
        }  
        return res;  
    }  
}
```
