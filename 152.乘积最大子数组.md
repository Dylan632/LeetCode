## 思路（必填）
### 算法选择
动态规划（滚动变量），同时维护最大乘积和最小乘积。
### 实现步骤
1. 初始化fMax/fMin为 1。
2. 每读取一个x，用上轮状态计算本轮最大/最小。
3. 用ans维护全局最大值。  
 

## 关键点
>[!Tip] 负负得正，所以最小值状态同样关键。

## 代码

```java
class Solution {  
    public int maxProduct(int[] nums) {  
        // dpMax[i]是以nums[i]结尾的乘积最大子数组  
        // dpMin[i]是以nums[i]结尾的乘积最小子数组  
        int[] dpMax = new int[nums.length];  
        int[] dpMin = new int[nums.length];  
        // 初始化：单个元素时最大和最小都是它自己  
        dpMax[0] = nums[0];  
        dpMin[0] = nums[0];  
        // 全局最大乘积  
        int maxProduct = nums[0];  
  
        for (int i = 1; i < nums.length; i++) {  
            // 关键逻辑：每个位置有三种可能  
            // 1. 前一个最大乘积 * 当前元素  
            // 2. 前一个最小乘积 * 当前元素（负负得正的情况）  
            // 3. 从当前位置开始（比如遇到0后或者前面是极小的负数）  
            int case1 = dpMax[i - 1] * nums[i];  
            int case2 = dpMin[i - 1] * nums[i];  
            int case3 = nums[i];  
            // 三者中取最大和最小  
            dpMax[i] = Math.max(Math.max(case1, case2), case3);  
            dpMin[i] = Math.min(Math.min(case1, case2), case3);  
            // 更新全局最大乘积  
            maxProduct = Math.max(maxProduct, dpMax[i]);  
        }  
        return maxProduct;  
    }  
}
```
## 优化后的代码
我们完全不需要开辟整个数组，只需要用几个变量（滚动变量）来记录上一轮的最大值和最小值即可。同时，我们可以在遍历的过程中顺便更新全局最大值，优化后的空间复杂度为O(1)。
```java
class Solution {  
    public int maxProduct(int[] nums) {  
        int ans = Integer.MIN_VALUE;  
        int fMax = 1;  
        int fMin = 1;  
        for (int x : nums) {  
            int preMax = fMax;  
            fMax = Math.max(x, Math.max(fMax * x, fMin * x));  
            // fMax已改变，所以使用preMax表示上一次循环的fMax  
            fMin = Math.min(x, Math.min(preMax * x, fMin * x));  
            ans = Math.max(ans, fMax);  
        }  
        return ans;  
    }  
}
```
