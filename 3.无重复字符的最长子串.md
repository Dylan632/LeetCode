## 思路（必填）

### 问题分析

- 目标：找出字符串中不含重复字符的最长子串的长度
- 关键词：“子串”（连续）、“无重复字符”、“最长”
- 约束：子串必须连续，不能跳过字符

### 算法选择

- 滑动窗口：维护一个可变长度的窗口 `[left, right]`，窗口内保证无重复字符
- 哈希表：记录字符的最新位置，O(1) 时间查找重复字符
- 双指针：左指针标记窗口起点，右指针扩展窗口，避免重复遍历

### 实现步骤

1. 初始化左指针 `left = 0`（窗口起点），哈希表记录字符索引
2. 右指针 `right` 从 0 遍历到 `len - 1`：
   - 若当前字符 `currentChar` 已在哈希表中：
     - 更新 `left` 为该字符上次出现索引的下一个位置 `dic.get(currentChar) + 1`
     - 使用 `Math.max` 确保 `left` 只能向右移动，不会回退
   - 将当前字符及其索引加入哈希表
   - 更新 `res = max(res, right - left + 1)`（闭区间窗口长度）
3. 返回 `res`

## 关键点

> [!Tip] 核心思想
> - 左指针初始值为 0：这种实现窗口为闭区间 `[left, right]`，长度计算为 `right - left + 1`
> - 使用 `Math.max` 更新左指针：避免左指针回退到窗口外的旧索引，这是核心技巧
> - 哈希表存储索引而非计数：可以直接定位到重复字符的位置，实现 O(1) 更新左指针

## 代码

```java
//leetcode submit region begin(Prohibit modification and deletion)  
class Solution {  
    public int lengthOfLongestSubstring(String s) {  
        Map<Character, Integer> dic = new HashMap<>();  
        int left = 0; // 窗口的左边界  
        int res = 0; // 记录最长字串的长度  
        int len = s.length();  
  
        for (int right = 0; right < len; right++) {  
            char currentChar = s.charAt(right);  
            // 如果遇到重复字符，且这个重复字符在当前窗口内  
            if (dic.containsKey(currentChar)) {  
                // 更新左指针 left ： 跳到重复字符上次出现的下一个位置  
                left = Math.max(left, dic.get(currentChar) + 1);  
            }  
            // 哈希表记录  
            dic.put(currentChar, right);  
            // 更新当前找到的最大长度 （右边界 - 左边界 + 1）  
            res = Math.max(res, right - left + 1);  
        }  
        return res;  
    }  
}  
//leetcode submit region end(Prohibit modification and deletion)
```

## 复杂度

- **时间复杂度**：O(n)，其中 n 为字符串长度
- **空间复杂度**：O(1)，字符的 ASCII 码范围为 0 ~ 127，哈希表最多使用 O(128) = O(1) 大小的额外空间
