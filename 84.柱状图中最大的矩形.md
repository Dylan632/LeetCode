## 题目
![[84.png]]
**提示**：

- `1 <= heights.length <=10^5`
- `0 <= heights[i] <= 10^4`
## 思路（必填）
### 问题分析
给定 n 个非负整数表示柱状图的高度，求能勾勒出的最大矩形面积。每个柱子的宽度为 1。
### 算法选择
使用单调栈。对于每根柱子，如果能快速找到它左边和右边第一个比它矮的柱子，就能确定以它为高的最大矩形宽度。
### 实现步骤
  1. 初始化 `left[]` 和 `right[]` 数组，分别记录每根柱子的左右边界
  2. 使用单调递增栈，一次遍历同时确定左右边界：
    - 当栈顶柱子高度 ≥ 当前高度时，栈顶柱子的右边界确定（就是当前柱子）
    - 当前柱子的左边界是栈顶元素（栈中保持递增，栈顶就是左边第一个更小的）
  3. 遍历每根柱子，计算 高度 × (右边界 - 左边界 - 1)，取最大值

## 关键点
>[!Tip] 
  >- 核心技巧是一次遍历同时计算左右边界：当出栈时确定右边界，入栈前确定左边界
  >- `right[]` 初始化为 n，处理右边没有更小柱子的情况
  >- `left[]` 使用 -1 表示左边没有更小柱子

## 代码

```java
class Solution {  
    public int largestRectangleArea(int[] heights) {  
        int n = heights.length;  
        int[] left = new int[n];  
        int[] right = new int[n];  
        Arrays.fill(right, n);  
        Deque<Integer> st = new ArrayDeque<>();  
        for (int i = 0; i < n; i++) {  
            int h = heights[i];  
            while (!st.isEmpty() && heights[st.peek()] >= h) {  
                right[st.pop()] = i;  
            }  
            left[i] = st.isEmpty() ? -1 : st.peek();  
            st.push(i);  
        }  
        int ans = 0;  
        for (int i = 0; i < n; i++) {  
            ans = Math.max(ans, heights[i] * (right[i] - left[i] - 1));  
        }  
        return ans;  
    }  
}
```

## 知识点

为了做到两次遍历，以及写法三的一次遍历，首先，把 `right[i]` 的定义略作修改，调整为：在 `i` 右侧的小于或等于 `h=heights[i]` 的最近元素的下标。

如果 `heights` 中没有相同的元素，这样修改不影响 `right[i]`。

如果 `heights` 中有相同的元素呢？比如 `heights=[1,3,4,3,2]`，左边那个 `3` 的 `right[i]` 会变小，导致矩形面积变小，这是否会导致计算错误？

不会。注意在这种情况下，这两个高为 `3` 的柱子，虽然（在写法二中）左边那个 `3` 的矩形面积变小了，但右边那个 `3` 的矩形面积是不变的，所以我们不会错过正确答案。

修改 `right[i]` 的定义后，我们可以把 `left` 和 `right` 合在一起计算，从而减少一次遍历：

在计算 `left` 的过程中，如果栈顶元素 `≥heights[i]`，那么 `i` 就是栈顶元素的 `right`。



## 复杂度

- Time: O(n)，其中 n 为 `heights` 的长度。每个元素入栈出栈各至多一次，所以二重循环是 O(n) 的。
- Space: O(n)
