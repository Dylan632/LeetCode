## 思路（必填）

### 问题分析
- 给定两个字符串，求它们的最长公共子序列（LCS）的长度
- 子序列不要求连续，但要求保持相对顺序
- 典型的双序列型动态规划问题

### 算法选择
- 选择动态规划：问题具有最优子结构和重叠子问题特性
- 状态定义：`dp[i][j]` 表示 text1 前 i 个字符与 text2 前 j 个字符的 LCS 长度

### 实现步骤
1. 初始化：创建 (n+1)×(m+1) 的 DP 数组，`dp[0][j]` 和 `dp[i][0]` 自动初始化为 0（空字符串的 LCS 为 0）
2. 状态转移：
   - 若 `s[i] == t[j]`：当前字符匹配，LCS 长度增加，`dp[i+1][j+1] = dp[i][j] + 1`
   - 若 `s[i] != t[j]`：当前字符不匹配，取删除 `s[i]` 或删除 `t[j]` 的较大值，`dp[i+1][j+1] = max(dp[i][j+1], dp[i+1][j])`
3. 返回结果：`dp[n][m]` 即为完整字符串的 LCS 长度

## 关键点

>[!Tip]
> - DP 数组下标偏移 1 位（i+1, j+1）避免边界判断
> - 状态转移方程体现了“选或不选”的决策：匹配则加入 LCS，不匹配则继承之前的最优解

## 代码

```java
class Solution {  
    public int longestCommonSubsequence(String text1, String text2) {  
        char[] s = text1.toCharArray();  
        char[] t = text2.toCharArray();  
        int n = s.length;  
        int m = t.length;  
        int[][] dp = new int[n+1][m+1];  
        for (int i = 0; i < n; i++) {  
            for (int j = 0; j < m; j++) {  
                dp[i + 1][j + 1] = s[i] == t[j] ? dp[i][j] + 1 : Math.max(dp[i][j + 1], dp [i + 1][j]);  
            }  
        }  
        return dp[n][m];  
    }  
}
```

## 知识点

```java

    for (int j = 0; j < m; j++) {
        // 核心状态转移方程
        dp[i + 1][j + 1] = s[i] == t[j] ? dp[i][j] + 1 : Math.max(dp[i][j + 1], dp [i + 1][j]);
    }
}
```

使用双重循环遍历两个字符串的所有字符组合。因为 `dp` 数组向右下方偏移了一位（加上了空字符串位），所以当前字符 `s[i]` 和 `t[j]` 对应的状态其实是 `dp[i+1][j+1]`。

**状态转移逻辑（三目运算符 `? :` 的部分）**：

1. **当 `s[i] == t[j]` （两个字符相同）**：
    - 说明找到了一个公共字符。
    - 那么当前的最长长度，等于“去掉这两个字符之前的最长长度”加上这个新找到的字符（也就是 1）。
    - **公式**： $dp[i+1][j+1] = dp[i][j] + 1$

2. **当 `s[i] != t[j]` （两个字符不相同）**：
    - 说明这两个字符不能同时出现在公共子序列的末尾。
    - 那我们就看两种情况，取它们的最大值：
        - 退一步看 `text1`：保留 `text2` 当前的字符，砍掉 `text1` 当前的字符，长度是 $dp[i][j+1]$。
        - 退一步看 `text2`：保留 `text1` 当前的字符，砍掉 `text2` 当前的字符，长度是 $dp[i+1][j]$。
    - **公式**： $dp[i+1][j+1] = \max(dp[i][j+1], dp[i+1][j])$


## 复杂度

- Time: O(nm)
- Space: O(nm)
