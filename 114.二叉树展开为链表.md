## 题目
![[114.png]]
**提示：**

- 树中结点数在范围 `[0, 2000]` 内
- `-100 <= Node.val <= 100`
## 思路（必填）
### 问题分析
题目要求将二叉树按**前序遍历**顺序原地展开为单链表。前序遍历顺序是 `1 -> 2 -> 3 -> 4 -> 5 -> 6`。
### 算法选择
**反向构建链表**的策略。如果从后往前看（`6 -> 5 -> 4 -> 3 -> 2 -> 1`），其实就是“右子树 -> 左子树 -> 根节点”的遍历顺序。
### 实现步骤
采用头插法构建链表，也就是从节点 6 开始，在 6 的前面插入 5，在 5 的前面插入 4，依此类推。
为此，要按照 6→5→4→3→2→1 的顺序访问节点。如何遍历这棵树，才能实现这个顺序？
按照右子树 - 左子树 - 根的顺序 DFS 这棵树。
DFS 的同时，记录当前链表的头节点为 head。一开始 head 是空节点。

1. 如果当前节点为空，返回。
2. 先遍历右子节点，把右子树变成右侧的一条单链表。
3. 再遍历左子节点，把左子树变成单链表，并“头插”到右子树链表的最前面。
4. 最后处理当前节点，把它的左子树置空，右指针指向刚才拼好的那一长串链表（`head`）。
5. 更新 `head` 为当前节点，继续向上层返回。
## 关键点
>[!Tip] 
>利用**“右-左-根”的逆向后序遍历**从树的末端开始，借助全局指针 `head` 采用**头插法**从后往前原地拼接，完美避开了修改指针导致的节点丢失问题。

## 代码

```java
class Solution {  
    private TreeNode head;  
    public void flatten(TreeNode root) {  
        if (root == null) {  
            return;  
        }  
        flatten(root.right);  
        flatten(root.left);  
        root.left = null;  
        // 头插法，相当于链表的 root.next = head        
        root.right = head;  
        // 现在链表头节点是 root        
        head = root;  
    }  
}
```

## 复杂度

- Time: O(n)，每个节点被访问且仅访问一次（$n$ 为树的节点数）。
- Space: O(n)，使用了系统调用栈进行递归，最坏情况（树退化为链表）递归深度为 $n$，平均情况（平衡二叉树）为 $O(\log n)$。
