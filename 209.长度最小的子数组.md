## 思路

### 问题分析
- 找到数组中和 ≥ target 的最短连续子数组
- 数组元素均为正整数（关键条件，保证前缀和单调递增）
- 若不存在满足条件的子数组，返回 0

### 算法选择
- 滑动窗口（双指针）：利用正整数特性，当 sum ≥ target 时收缩左窗口
- 相比暴力枚举 O(n²) 和前缀和+二分查找 O(n log n)，滑动窗口可达 O(n)

### 实现步骤
1. 初始化左指针 left、窗口和 sum、最小长度 ans = n + 1
2. 右指针 right 遍历数组，将 nums[right] 加入窗口
3. 当窗口和 sum ≥ target 时：
   - 更新最小长度 ans = min(ans, right - left + 1)
   - 收缩左边界，sum -= nums[left]，左指针右移
4. 返回结果（若 ans ≤ n 则返回 ans，否则返回 0）

### 关键点
> [!Tip]
> - 使用 n + 1 作为初始值比 Integer.MAX_VALUE 更安全（避免溢出）
> - 正整数性质保证：右指针右移 → sum 增大，左指针右移 → sum 减小
> - 双指针只需各遍历一次数组，时间复杂度 O(n)

## 代码

```java
class Solution {
    public int minSubArrayLen(int target, int[] nums) {
        int n = nums.length;
        int ans = n + 1;
        int sum = 0;
        int left = 0;
        for (int right = 0; right < n; right++) {
            sum += nums[right];
            while (sum >= target) {
                ans = Math.min(ans, right - left + 1);
                sum -= nums[left];
                left++;
            }
        }
        return ans <= n ? ans : 0;
    }
}
```
