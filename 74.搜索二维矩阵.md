## 题目

![[74.png]]

**提示：**

- `m == matrix.length`
- `n == matrix[i].length`
- `1 <= m, n <= 100`
- `-10^4 <= matrix[i][j], target <= 10^4`

## 思路

### 问题分析

题目给定的二维矩阵有两个重要特性：每行元素严格单调递增，且每行的第一个整数大于前一行的最后一个整数。这构成了全局递增的条件。

### 算法选择

二分查找。因为全局严格递增，所以无需逐行或逐个遍历，直接利用二分查找可以将时间复杂度从 O(m * n) 降维打击到 O(log(m * n))。

### 实现步骤
1. 将二维矩阵虚拟化为一个长度为 `m * n` 的一维数组，设定左右双指针 `left` 和 `right`。
2. 在 `while (left <= right)` 的循环中，计算中间索引 `mid`。
3. **关键转换**：将一维的 `mid` 索引还原回二维坐标。所在行为 `mid / n`，所在列为 `mid % n`。
4. 比较该坐标对应的值与 `target` 的大小，进而不断缩小搜索区间，直到找到目标值或区间被压扁为空。

## 关键点

> [!Tip] 二维转一维的坐标映射
> `a[i] = matrix[i / n][i % n]`
> 一维坐标到二维坐标的映射公式 `row = index / cols` 和 `col = index % cols` 是解这道题的灵魂。

## 代码（二分查找）

```java
class Solution {  
    public boolean searchMatrix(int[][] matrix, int target) {  
        int m = matrix.length;  
        int n = matrix[0].length;  
        int left = 0;  
        int right = m * n - 1;  
        while (left <= right) {  
            int mid = left + (right - left) / 2;  
            int x = matrix[mid / n][mid % n];  
            if (x == target) {  
                return true;  
            }  
            if (x < target) {  
                left = mid + 1;  
            } else {  
                right = mid - 1;  
            }  
        }  
        return false;  
    }  
}
```

## 代码（矩阵排除法）

```java
class Solution {
    public boolean searchMatrix(int[][] matrix, int target) {
        int i = 0;
        int j = matrix[0].length - 1;
        while (i < matrix.length && j >= 0) { // 还有剩余元素
            if (matrix[i][j] == target) {
                return true; // 找到 target
            }
            if (matrix[i][j] < target) {
                i++; // 这一行剩余元素全部小于 target，排除
            } else {
                j--; // 这一列剩余元素全部大于 target，排除
            }
        }
        return false;
    }
}
```

## 知识点

由于矩阵的每一行是递增的，且每行的第一个数大于前一行的最后一个数，如果把矩阵每一行拼在一起，我们可以得到一个递增数组。

![[74_1.png]]

例如示例 1，三行拼在一起得：

`a = [1, 3, 5, 7, 10, 11, 16, 20, 23, 30, 34, 60]`

由于这是一个有序数组，我们可以用二分查找判断 target 是否在 matrix 中。

代码实现时，并不需要真的拼成一个长为 `m * n` 的数组 `a`，而是将 `a[i]` 转换成矩阵中的行号和列号。例如示例 1，`i = 9` 对应的 `a[i] = 30`，由于矩阵有 `n = 4` 列，所以 `a[i]` 在 `i / n = 2` 行，在 `i % n = 1` 列。

一般地，有：`a[i] = matrix[i / n][i % n]`

## 复杂度

### 二分查找

- Time: O(log(m * n))
- Space: O(1)
其中 m 和 n 分别为 matrix 的行数和列数。
### 矩阵排除法

- Time: O(m + n)
- Space: O(1)
